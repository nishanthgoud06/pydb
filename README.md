# PyDB

A mini database engine built from scratch in Python.

## What this is

A learning project to understand how databases work under the hood —
storage engines, query parsing, indexing, and concurrency.

## Phases

- [x] Phase 1: Storage engine (pager, row serialization, table)
- [ ] Phase 2: Query parser
- [ ] Phase 3: Query executor
- [ ] Phase 4: Concurrency layer

## Running it

```bash
python main.py
```

## Stack

- Python 3.11
- No external dependencies
